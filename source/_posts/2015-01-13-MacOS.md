### 8. pbcopy

> 使用 **pbcopy** 命令,对应有个 **pbpaste** 命令。

示例：

```Bash
echo 'Hello World!' | pbcopy
```

_将myFile.txt中读取的字符串Hello World放入剪贴板_

```Bash
cat myFile.txt | pbcopy
```

```Bash
pbpaste > file.txt
```

### 9. say

spotlight搜索框中输入"终端"或者"terminal", 找到并打开"终端"这个程序。

```bash
say hello world!
```

> **mac say 唱歌**

打开[此链接](<[https://www.macx.cn/thread-2048013-1-1.html?mod=viewthread&tid=2048013&extra=page%253D1&page=1](https://www.macx.cn/thread-2048013-1-1.html?mod=viewthread&tid=2048013&extra=page%3D1&page=1)>),cmd+c复制歌词，cmd+v粘贴到终端窗口，回车。
mac就开始开始听歌了。
