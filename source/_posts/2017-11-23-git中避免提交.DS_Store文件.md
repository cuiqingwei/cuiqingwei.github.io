### 1. 先删除原有的.DS_Store：

```
find . -name ".DS_Store" -type f -delete
```

命令解释：在当前文件夹以及当前文件夹的子文件夹中找到所有的.DS_Store文件

### 2. 建立.gitignore文件

vi .gitignore。然后添加.DS_Store作为忽略：

```
.DS_Store
```

### 3. 提交到git

```
git add .gitignore
git commit -m 'delete .DS_Store'
```
