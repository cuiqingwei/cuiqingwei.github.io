> 今年的上海，注定是不平凡的一年。疫情期间，大家只能居家办公。我也在居家办公的同时，不断优化和提升MAC生产力。

Do one thing, and do it well. - A principle of Unix philosophy

### 一些提升效率的建议：

- 指法用对，掌握盲打，提升敲击数字键的成功率;

- 多用键盘快捷键少用鼠标。修改系统键盘按键，比如 `Caps Lock`⇪ 使用频率比较低，一般我改成了 `Ctrl`⌃键，因为用开发工具频繁使用到 Ctrl，改成 Ctrl 之后效率提高不少，小指头不酸啦（推荐通过系统设置-键盘-修饰键盘…修改）;

  [Karabiner-Elements](https://github.com/pqrs-org/Karabiner-Elements)

  [搜索键值配置](https://ke-complex-modifications.pqrs.org/)
  - Ctrl 与 Caps Lock 互换

  - Map ctrl + [ to escape，Change Fn + h/j/k/l to Arrows

- 用好终端和命令行工具。

## 效率软件

- Alfred3 ：[Alfred](https://www.alfredapp.com/) 是 Mac 上一款著名的效率应用，强大的功能和众多的扩展能让你在实际操作中大幅提升工作效率。基本上打开软件，开启、关闭电脑都可以，还可以通过手机客户端app远程控制，如果mac电脑只选择一个软件，我肯定会选择它；

- [Rectangle](https://github.com/rxhanson/Rectangle)：窗口管理辅助工具，拖拽窗口到屏幕边缘可以自动半屏， 全屏或者四分之一屏幕，都可以通过快捷键完成分屏；

- [vimac](https://github.com/dexterleng/vimac): 用 vim 的方式操作 mac（`Ctrl+F`打开快捷键提示，`Ctrl+J`聚焦窗口vim命令滚动），手指不用移开键盘就可以操作屏幕，提升效率！

- [Notion](https://www.notion.so/zh-cn)：笔记软件；

- [Hidden Bar](https://github.com/dwarvesf/hidden)：一个超轻MacOS实用工具，帮助隐藏菜单栏图标。

## 终端阵容

- `iTerm2` 替代原生终端；

- 安装 `oh-my-zsh`以及相关插件，通过homebrew安装管理软件；

  [cuiqingwei/dotfiles](https://github.com/cuiqingwei/dotfiles)

- [Tabby](https://github.com/Eugeny/tabby)：终端的终点站；

- [Warp](https://www.warp.dev/)：21世纪AI终端；

- 快捷键

  Ctrl + a 光标移动到行首 Ctrl + e 光移动到行尾 Ctrl + f 光标右移1个字符 Ctrl + b 光标左移一个字符 Ctrl + r 搜索history中的命令;

- [tmux](https://github.com/gpakosz/.tmux) **:** Oh my tmux! 是一款优秀的终端复用软件。

## 网络利器

- [Tunnelblick](https://github.com/Tunnelblick/Tunnelblick) ：software for OpenVPN on macOS；

- [Motrix](https://github.com/agalwood/Motrix)：A full-featured download manager。

## Chrome 插件

- **Momentum**: 美化 Chrome 界面的插件，有很多漂亮的图片作为背景；
- **Octotree**: Chrome GitHub 浏览插件，可以把 GitHub 项目的目录树结构展示出来，还可以侧边隐藏，非常实用；
- **Adblock**：最近广告拦截工具；
- **Neater Bookmarks**：用了多年的书签管理插件；
- **crxMouse Chrome™ 手势**：通过鼠标手势控制操作网页；
- **SurfingKeys**/Vimium：可以用 vim 的方式操作浏览器，不用鼠标也能完成大部分操作，新秀SurfingKeys比较强大，可以用 Ctrl+i 来用 vim 编辑输入；
- **Tampermonkey**：油猴；
- 购物党自动比价工具。
